    源码解析之:gin #

    gin 是一个用golang实现的HTTPweb框架。

    特性 #

    官网上描述，gin的特性包括：

    • 快：路由使用基数树，低内存，不使用反射；
    • 中间件注册：一个请求可以被一系列的中间件和最后的action处理
    • 奔溃处理：gin可以捕获panic使应用程序可用
    • JSON校验：将请求的数据转换为JSON并校验
    • 路由组：更好的组织路由的方式，无限制嵌套而不影响性能
    • 错误管理：可以收集所有的错误
    • 内建渲染方式：JSON，XML和HTML渲染方式
    • 可继承：简单的去创建中间件
