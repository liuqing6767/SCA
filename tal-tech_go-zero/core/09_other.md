# 其他

| 包       | 说明                                                  |
| -------- | ----------------------------------------------------- |
| cmdline  | 命令行接收相关的工具函数                              |
| codec    | 加解密/编码相关的二次封装                             |
| conf     | yaml/json加解密, kv 的序列化和反序列化                |
| contextx | context 的二次封装                                    |
| errorx   | error相关的二次封装，包括多个error和原子化操作的error |
| filex    | 文件相关的封装，比如读取文件第一行/最后一行           |
